Training the MNIST Handwriting Recognition Model

Step 1: Install Dependencies

Make sure you have all the required dependencies installed:

pip install -r requirements.txt

Step 2: Train the Model

Run the training script to download the MNIST dataset and train the model: 

python mnist_handwriting_recognition.py

This will:

  • Download the MNIST dataset (if not already downloaded)
  • Preprocess the data
  • Create and train a convolutional neural network
  • Save the trained model to models/mnist_model.h5
  • Generate performance visualizations

Note: The training process may take several minutes depending on your hardware. The script will display progress information during training.

Step 3: Use the Web Application

Once the model is trained, you can use the web application to draw digits and get predictions: 

python app.py

Then open your browser and navigate to: http://127.0.0.1:5000

Step 4: Test with Custom Images (Optional)

You can also test the model with your own image files:

python test_custom_image.py path/to/your/image.png

The image should contain a single handwritten digit on a contrasting background.

Important: The model is trained on the MNIST dataset, which consists of 28x28 pixel grayscale images of handwritten digits (0-9). For best results, your handwritten digits should be:

  • Centered in the drawing area
  • Large enough to fill a significant portion of the drawing area
  • Written with a thick stroke
  • Written on a contrasting background
